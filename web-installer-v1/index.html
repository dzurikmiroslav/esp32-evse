esp32-evse installer

  1. Connect the EVSE board to your computer using USB or serial-to-USB adapter
  2. Press Connect button and select connet serial port
  3. Confirm install in dialog, some boards must be manualy switched to flashing mode by holding WiFi buton while reset (ESP32-S2 EVSE DIY ALPHA)
  4. Install firmware
  5. Set up a WiFi network in dialog, if none is set up
  6. Then click Visit Device in dialog, should open Web interface of EVSE board
  7. Go to settings / files
  8. Then modify board.yaml and restart device
Installer powered by ESP Web Tools.
